Open-source homebrew utility for PlayStation 3 focused on HDD analysis and storage management.

---

## 🚀 About the Project

Mnemosyne HDD Tool is a work-in-progress PS3 homebrew application designed to help users understand and manage their console storage in a simple and safe way.

This project is currently in **early development stage**, and the main goal is to build a solid foundation for future features.

---

## 📌 Planned Features

- 📊 HDD storage analysis
- 📁 File and folder usage visualization
- 🧹 Safe cleanup tools (no system risk)
- ⚡ Lightweight performance-focused design
- 🎮 PS3 homebrew integration (.pkg builds planned)

---

## ⚠️ Status

🚧 **Work in progress**

The toolchain and core system are still being set up.  
No stable release is available yet.

Expect frequent changes and experimental builds.

---

## 🛠️ Tech Stack (planned)

- C (PSL1GHT / PS3 toolchain)
- Makefiles
- PS3 SDK homebrew environment

---

## 📦 Goal

The final goal is to produce a reliable and safe HDD management tool that runs directly on PlayStation 3 consoles.
