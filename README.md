## About

Mnemosyne HDD Tool is an open-source PlayStation 3 homebrew project focused on providing an intuitive, modern and safe interface for storage analysis and HDD management.

The project aims to replace the traditional "file browser only" approach with a visual dashboard capable of displaying storage usage, directory statistics and useful system information.

The software is currently under active development.

---

## Current Status

> **Early Development**

The project architecture and user interface are currently being developed.

Hardware access, filesystem analysis and HDD scanning will be implemented during future milestones.

No stable release is available yet.
